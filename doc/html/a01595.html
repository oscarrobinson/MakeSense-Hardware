Detailed Description

Driver for the MS5611-01BA03 barometric pressure sensor in I2C mode.

We cannot easily do the calculation needed to transform the raw values into calibrated values because the numbers get too large to hold in standard C types for this compiler. In short, we need 64-bit integers in which to do the calculations. But even long longs are only 32 bits on this platform.

Instead we will allow a client to obtain the config values and the raw data to allow the calculation to be done elsewhere.
