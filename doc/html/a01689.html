Detailed Description

Disco server sources (embedded part of the DISCOBALL project)

It objective is to receive a code file over UDP, store it in external flash and disseminate it to other nodes of the 6LoWPAN network.

For this to work, the image must be co-hosted with the BooTTY! bootloader, which will move the image from external to internal flash.

To link this application in your contiki image, all you need to do is to add this line: OFFSET_FIRMWARE=1 to your project's makefile
